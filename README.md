# 📝 Simulado ENEM Interativo

Simulado completo no formato ENEM com 128 questões no banco.

## ✨ Funcionalidades

- 🎯 32 questões por simulado (8 por área)
- 📊 Correção pelo método TRI
- 🏆 Ranking persistente (localStorage)
- 🔍 Detecção de padrão de chute
- 📱 Interface responsiva
- ⏱️ Controle de tempo com alerta
